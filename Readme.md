- Check all templates
```c#
//1.
dotnet new --list 
// or 2.
dotnet new -l
```

```c#
// 1.
// create a empty project folder
//in this folder cmd/terminal
dotnet new short_name_of_template

//2.
dotnet new short_name_of_template -o project_name
```
- run the project
```c#
// first go in project folder
dotnet watch run
```
